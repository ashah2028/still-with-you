# Still With You

Still With You is a nonprofit web platform that helps patients feel emotionally connected to their loved ones during long hospital stays.

## Tech Stack
- Frontend: React + TypeScript (Vite)
- Backend: Spring Boot
- Database: PostgreSQL
- Storage: AWS S3

## Repo Structure
- /frontend – React web app
- /backend – Spring Boot API
- /docs – planning and documentation

## Status
Early MVP under active development.
